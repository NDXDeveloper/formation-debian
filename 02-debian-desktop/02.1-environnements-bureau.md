# 2.1 Environnements de bureau

## 2.1.1 GNOME (environnement par défaut)

### Qu'est-ce que GNOME ?

**GNOME** (GNU Network Object Model Environment) est l'environnement de bureau **par défaut** de Debian depuis de nombreuses années. Il représente la philosophie moderne du desktop Linux avec une approche épurée et intuitive.

**Caractéristiques principales :**
- **Design moderne** : Interface épurée et minimaliste
- **Simplicité d'usage** : Conçu pour être intuitif pour les débutants
- **Intégration complète** : Applications cohérentes entre elles
- **Accessibilité** : Excellent support des technologies d'assistance
- **Écosystème riche** : Nombreuses applications et extensions

### Interface et navigation GNOME

#### Le bureau GNOME

**Vue d'ensemble :**
```
┌────────────────────────────────────────────────────┐
│  [Activités]                    [🔊][📶][🔋][⚙️]  │ ← Barre supérieure
├────────────────────────────────────────────────────┤
│                                                    │
│              Bureau propre                         │
│          (pas d'icônes par défaut)                 │
│                                                    │
│                                                    │
└────────────────────────────────────────────────────┘
```

**Éléments principaux :**
- **Barre supérieure** : Toujours visible, accès aux fonctions système
- **Bureau vide** : Philosophie "focus sur les applications"
- **Dock caché** : Apparaît en mode "Activités"

#### Mode Activités (Overview)

**Accès au mode Activités :**
- **Clic** sur "Activités" (coin supérieur gauche)
- **Touche Super** (touche Windows)
- **Coin actif** : Déplacer la souris au coin supérieur gauche

**Vue Activités :**
```
┌────────────────────────────────────────────────────┐
│ [🔍 Rechercher...]                    [Bureau 1][+]│
├────────────────────────────────────────────────────┤
│  ┌─────────┐  ┌─────────┐  ┌─────────┐            │
│  │Fenêtre 1│  │Fenêtre 2│  │Fenêtre 3│            │ ← Fenêtres ouvertes
│  └─────────┘  └─────────┘  └─────────┘            │
├────────────────────────────────────────────────────┤
│ [📁][🌐][📧][⚙️][📝][🎵][🎬][📷]                 │ ← Dock applications
└────────────────────────────────────────────────────┘
```

#### Navigation par espaces de travail

**Espaces de travail (Workspaces) :**
- **Concept** : Bureaux virtuels pour organiser les applications
- **Création automatique** : Nouveaux espaces créés selon les besoins
- **Navigation** :
  - **Super + Page Up/Down** : Changer d'espace
  - **Ctrl + Alt + Flèches** : Navigation alternative
  - **Glisser-déposer** : Déplacer fenêtres entre espaces

### Personnalisation GNOME

#### GNOME Tweaks (Réglages)

**Installation :**
```bash
sudo apt install gnome-tweaks
```

**Fonctionnalités principales :**
- **Apparence** : Thèmes, icônes, curseurs
- **Bureau** : Icônes sur le bureau, corbeille
- **Extensions** : Activation/désactivation
- **Polices** : Configuration typographique
- **Clavier** : Mappings et raccourcis
- **Fenêtres** : Comportement et boutons

#### Extensions GNOME

**Qu'est-ce que c'est ?**
Les **extensions** sont des **modules complémentaires** qui ajoutent des fonctionnalités à GNOME.

**Installation depuis le navigateur :**
1. **Visitez** https://extensions.gnome.org/
2. **Installez** l'extension navigateur GNOME Shell
3. **Parcourez** et installez les extensions désirées
4. **Gérez** via GNOME Tweaks

**Extensions populaires :**

| Extension | Fonctionnalité | Utilité |
|-----------|---------------|---------|
| **Dash to Dock** | Dock permanent | Accès rapide applications |
| **User Themes** | Thèmes personnalisés | Personnalisation poussée |
| **Applications Menu** | Menu traditionnel | Navigation familière |
| **Places Status Indicator** | Accès dossiers rapide | Efficacité |
| **Weather** | Météo dans la barre | Information utile |
| **Night Light Slider** | Contrôle lumière bleue | Confort visuel |

### Avantages et inconvénients GNOME

#### Avantages ✅

- **Simplicité** : Interface intuitive pour débutants
- **Cohérence** : Applications uniformes et intégrées
- **Modernité** : Design contemporain et épuré
- **Accessibilité** : Excellent support handicaps
- **Stabilité** : Environnement mature et testé
- **Écosystème** : Nombreuses applications natives
- **Documentation** : Ressources abondantes

#### Inconvénients ❌

- **Consommation ressources** : Plus lourd que des alternatives
- **Personnalisation limitée** : Moins flexible par défaut
- **Workflow imposé** : Philosophie particulière à adopter
- **Extensions fragiles** : Compatibilité versions GNOME
- **Pas d'icônes bureau** : Par défaut (modifiable)

**Recommandé pour :**
- ✅ **Débutants Linux** cherchant simplicité
- ✅ **Utilisateurs productivité** privilégiant efficacité
- ✅ **Écrans tactiles** et usage moderne
- ✅ **Intégration cloud** et services en ligne

## 2.1.2 KDE Plasma

### Qu'est-ce que KDE Plasma ?

**KDE Plasma** est un environnement de bureau **hautement personnalisable** développé par la communauté KDE. Il offre une expérience riche en fonctionnalités avec une philosophie de **liberté totale** de personnalisation.

**Philosophie KDE :**
- **Puissance** : Fonctionnalités avancées et complètes
- **Personnalisation** : Contrôle total sur l'apparence et comportement
- **Traditionnel** : Interface familière aux utilisateurs Windows
- **Communauté** : Développement collaboratif international

## 2.1.3 XFCE et LXDE

### XFCE - L'équilibre parfait

#### Qu'est-ce que XFCE ?

**XFCE** (XForms Common Environment) est un environnement de bureau **léger** qui privilégie la **performance** tout en conservant une interface **complète** et **intuitive**.

**Philosophie XFCE :**
- **Légèreté** : Consommation ressources minimale
- **Rapidité** : Réactivité sur ancien matériel
- **Simplicité** : Interface claire sans superflu
- **Stabilité** : Code mature et fiable
- **Modularité** : Composants indépendants

#### Interface XFCE

**Vue d'ensemble :**
```
┌────────────────────────────────────────────────────┐
│                                                    │
│  [📁] [🗎]     Bureau traditionnel             [🗑️] │
│                                                    │
│  [⚙️Applications]                                  │
│                                                    │
└────────────────────────────────────────────────────┘
│[Menu][📁][🌐] Applications    [14:30][🔊][📶][🔋]│ ← Panel
└────────────────────────────────────────────────────┘
```

**Composants principaux :**
- **Panel** : Barre des tâches configurable
- **Menu Whisker** : Lanceur d'applications moderne
- **Gestionnaire fenêtres** : Xfwm4, léger et efficace
- **Bureau** : Xfdesktop avec icônes et fond d'écran

#### Applications XFCE

**Suite d'applications natives :**

| Application | Nom technique | Usage |
|-------------|---------------|-------|
| **Thunar** | thunar | Gestionnaire de fichiers |
| **Terminal** | xfce4-terminal | Émulateur terminal |
| **Text Editor** | mousepad | Éditeur simple |
| **Application Finder** | xfce4-appfinder | Recherche applications |
| **Task Manager** | xfce4-taskmanager | Gestionnaire tâches |
| **Settings Manager** | xfce4-settings | Configuration |

#### Personnalisation XFCE

**Settings Manager :**
- **Appearance** : Thèmes et icônes
- **Window Manager** : Comportement fenêtres
- **Panel** : Configuration barre tâches
- **Desktop** : Fond d'écran et icônes
- **Keyboard** : Raccourcis personnalisés

**Avantages XFCE :**
- ✅ **Très léger** : RAM < 512 MB
- ✅ **Rapide** : Démarrage ultra-rapide
- ✅ **Stable** : Peu de bugs
- ✅ **Familier** : Interface traditionnelle
- ✅ **Personnalisable** : Équilibre simplicité/options
- ✅ **Mature** : Développement stable

### LXDE/LXQt - L'ultra-léger

#### Qu'est-ce que LXDE ?

**LXDE** (Lightweight X11 Desktop Environment) est l'environnement **le plus léger** disponible, conçu pour les **très anciens ordinateurs** et les **systèmes à ressources limitées**.

**Caractéristiques LXDE :**
- **Minimalisme extrême** : Fonctions essentielles uniquement
- **Consommation minimal** : RAM < 300 MB
- **Démarrage instantané** : Quelques secondes
- **Compatibilité étendue** : Fonctionne sur très vieux matériel

#### LXQt - L'évolution moderne

**LXQt** est la **version moderne** de LXDE, basée sur Qt au lieu de GTK.

**Améliorations LXQt :**
- **Interface modernisée** : Look plus contemporain
- **Technologies récentes** : Qt5/Qt6, Wayland
- **Meilleure intégration** : Applications Qt natives
- **Développement actif** : Mises à jour régulières

#### Applications LXDE/LXQt

**Suite minimaliste :**

| Application | Usage | Caractéristiques |
|-------------|-------|------------------|
| **PCManFM** | Gestionnaire fichiers | Ultra-rapide, onglets |
| **LXTerminal** | Terminal | Léger, configurable |
| **Leafpad** | Éditeur texte | Minimal, efficace |
| **LXTask** | Gestionnaire tâches | Simple, fonctionnel |

#### Quand choisir LXDE/LXQt ?

**Situations idéales :**
- **Très ancien matériel** : < 2 GB RAM, CPU < 1 GHz
- **Netbooks** et mini-PC
- **Serveurs** nécessitant GUI minimale
- **Systèmes embarqués** avec contraintes ressources
- **Performance maximale** prioritaire

**Limitations :**
- ❌ **Interface basique** : Peu d'effets visuels
- ❌ **Applications limitées** : Suite logicielle réduite
- ❌ **Personnalisation réduite** : Options limitées
- ❌ **Moins moderne** : Look années 2000

## 2.1.4 Installation et configuration des DE

### Préparation à l'installation

#### Vérification du système actuel

**Identifier l'environnement installé :**
```bash
# Voir l'environnement de bureau actuel
echo $XDG_CURRENT_DESKTOP

# Voir tous les environnements installés
ls /usr/share/xsessions/

# Vérifier les gestionnaires de connexion
systemctl status gdm3 sddm lightdm
```

#### Considérations importantes

**⚠️ Points d'attention avant installation :**
- **Espace disque** : Chaque DE nécessite 1-3 GB
- **Conflits potentiels** : Applications similaires de différents DE
- **Gestionnaire de connexion** : Chaque DE a son preference
- **Cohérence** : Mélanger les DE peut créer confusion

**Recommandations :**
- ✅ **Créer utilisateur test** : Tester sans impacter config principale
- ✅ **Sauvegarde configuration** : `.config/` et `.local/`
- ✅ **Notes personnalisations** : Documenter modifications

### Installation GNOME

#### Installation complète

**Si GNOME pas installé :**
```bash
# Mise à jour système
sudo apt update

# Installation complète GNOME
sudo apt install gnome-core gnome-tweaks

# Installation des applications (optionnel)
sudo apt install gnome-games gnome-weather gnome-music

# Gestionnaire de connexion GNOME
sudo apt install gdm3
```

#### Installation minimale

**Pour installation légère :**
```bash
# GNOME minimal
sudo apt install gnome-shell gnome-terminal nautilus

# Applications essentielles
sudo apt install gnome-control-center gnome-tweaks
```

#### Configuration post-installation

**Premier démarrage :**
1. **Déconnectez-vous** de la session actuelle
2. **Sélectionnez** "GNOME" dans l'écran de connexion
3. **Connectez-vous** avec vos identifiants
4. **Suivez** l'assistant initial setup

**Configuration recommandée :**
```bash
# Extensions utiles
sudo apt install gnome-shell-extensions

# Support thèmes
sudo apt install gnome-themes-extra

# Polices supplémentaires
sudo apt install fonts-noto fonts-liberation
```

### Installation KDE Plasma

#### Installation complète

**Installation KDE Desktop :**
```bash
# Mise à jour système
sudo apt update

# Installation complète KDE
sudo apt install kde-full

# Ou installation plus légère
sudo apt install kde-standard
```

#### Installation minimale

**Pour système léger :**
```bash
# KDE minimal
sudo apt install kde-plasma-desktop

# Applications essentielles
sudo apt install dolphin konsole kate
```

#### Gestionnaire de connexion

**SDDM (recommandé pour KDE) :**
```bash
# Installation SDDM
sudo apt install sddm

# Configuration comme gestionnaire par défaut
sudo dpkg-reconfigure sddm

# Ou configuration manuelle
sudo systemctl enable sddm
sudo systemctl disable gdm3
```

#### Configuration KDE

**Première connexion :**
1. **Sélectionnez** "Plasma" dans l'écran de connexion
2. **Explorez** System Settings
3. **Configurez** apparence selon préférences

**Paquets recommandés :**
```bash
# Codecs multimédia
sudo apt install ubuntu-restricted-extras

# Applications supplémentaires
sudo apt install krita kdenlive k3b

# Support formats propriétaires
sudo apt install libavcodec-extra
```

### Gestion des gestionnaires de connexion

#### Changer de gestionnaire

**Commande de reconfiguration :**
```bash
# Reconfigurer le gestionnaire par défaut
sudo dpkg-reconfigure gdm3
sudo dpkg-reconfigure sddm
sudo dpkg-reconfigure lightdm
```

**Configuration manuelle :**
```bash
# Désactiver gestionnaire actuel
sudo systemctl disable gdm3

# Activer nouveau gestionnaire
sudo systemctl enable sddm
sudo systemctl start sddm
```

#### Comparaison gestionnaires

| Gestionnaire | DE associé | Ressources | Fonctionnalités |
|--------------|------------|------------|-----------------|
| **GDM3** | GNOME | Élevées | Modernes, Wayland |
| **SDDM** | KDE | Moyennes | Thèmes, flexible |
| **LightDM** | XFCE/LXDE | Faibles | Léger, stable |

### Résolution des problèmes

#### Conflits entre DE

**Problèmes courants :**
- **Applications multiples** pour même fonction
- **Thèmes incohérents**
- **Raccourcis en conflit**
- **Services redondants**

**Solutions :**
```bash
# Nettoyer applications orphelines
sudo apt autoremove

# Reset configuration utilisateur
mv ~/.config ~/.config.backup
mv ~/.local ~/.local.backup

# Nouvelle session propre
```

#### Retour environnement précédent

**Si problème avec nouveau DE :**
1. **Écran de connexion** : Sélectionner ancien DE
2. **Désinstallation** : `sudo apt remove kde-full` (exemple)
3. **Nettoyage** : `sudo apt autoremove`
4. **Reset configuration** utilisateur si nécessaire

### Optimisation post-installation

#### Configuration commune

**Applications tierces populaires :**
```bash
# Navigation web
sudo apt install firefox-esr chromium

# Multimédia
sudo apt install vlc gimp audacity

# Bureautique
sudo apt install libreoffice

# Développement
sudo apt install git vim code
```

#### Personnalisation finale

**Étapes recommandées :**
1. **Configuration** fond d'écran et thème
2. **Installation** applications spécifiques besoins
3. **Raccourcis** clavier personnalisés
4. **Sauvegarde** configuration finale

**Documentation configuration :**
```bash
# Créer documentation personnelle
echo "# Ma configuration Debian Desktop" > ~/config-notes.md
echo "Environment: $XDG_CURRENT_DESKTOP" >> ~/config-notes.md
echo "Date: $(date)" >> ~/config-notes.md
```

L'installation et configuration des environnements de bureau vous permet maintenant de choisir l'interface qui correspond le mieux à vos besoins et contraintes matérielles.
